1.cd	
切换目录
例如：
cd /root/Documents

2.ls
查看当前目录下的文件
-l	列出每个文件的权限等信息

3.pwd	
查看当前目录的路径	

4.sudo
提权
sudo -i

5.du

6.mkdir（英文全拼：make directory）
用于在当前路径下新建目录(文件夹)
  (1)mkdir Name
（2）mkdir -p Name 01 02		一次实现多级创建（若一级目录存在，则在一级目录下创建；若一级目录不存在，则创建并命名一个这样的一级目录。）
（3）mkdir -m 777 Name		创建并设置该目录的权限（如仅读）

7.touch
在当前目录下新建（不存在）或修改（已存在）文件

8.echo
往文件中写入内容

9.cp（英文全拼：copy file）
用于复制文件或路径
(1)cp [options] source dest
    cp 选项 源文件 目标文件
（2）选项参数说明
-d	复制时保留链接(这里的链接相当于Windows中的快捷方式)

10.rm(英文全拼：remove)

用于删除一个文件或目录
（1）参数说明
-i	删除前逐一询问确认
-f	直接删除，不需确认（即使文件属性为只读）
（2）删除当前文件夹下所有文件：rm -r *

11.mv（英文全拼：move）
用于移动文件或目录（或者改名。但实际上，“改名”操作的本质，是重新创建一个名字不同，但内容完全一致的文件，再将原文件删除。
（1）mv 原名称 新名称

12.which
用于查找文件（在环境变量$PATH设置的目录中查找符合要求的文件
（1）which ifconfig

13.find
用于在指定目录下查找目录和文件
find【路径】【匹配条件】【动作】
（1）匹配条件
-name	按文件名查找
-type	按文件类型查找
（2）路径
（可以是一个或多个路径，多个路径之间用空格分隔
若缺省，则默认为当前目录
（3）动作
-amin n		查找在n分钟内被访问过的文件
-atime n		查找在n*24小时内访问过的文件

14.chmod(英文全拼：change mode）
设置用户对文件的权限

15.查看进程
(1).ps
查看静态的进程统计信息
ps aux

(2).pstree
以树状图形式显示进程信息

(3).top
查看实时更新的动态进程信息

(4).pgrep
根据相关条件查询特定进程的PID



16.终止进程
(1)kill
使用进程的PID作为参数
若该进程无法响应，则可用-9强行终止

(2)killall
以进程名为参数
若该进程无法响应，则可用-9强行终止

(3)pkill
可根据进程名，运行该进程的用户，进程所在终端等多种属性终止特定的进程
